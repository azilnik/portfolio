Blockdaemon is a blockchain infrastructure platform. What Twilio did for the tech industry to text and call via API is what Blockdaemon does to enable any company to connect to blockchains.

I joined Blockdaemon as the first designer, helping design their web and mobile apps. These apps allow users to access real-time data across all their different blockchain assets in one place.

The workflows enable users to start monitoring wallets, create new portfolios, and generate reports that can be used for tax purposes and for reporting to management and clients.
