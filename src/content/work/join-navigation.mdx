Join is a project management platform that helps construction companies, owners, and architects make better, more informed decisions.

As new features have been added to Join, the app's breadth has made critical features undiscoverable, and customers have cited workflows as overly cumbersome.

This project simplified our navigation by revisiting the architecture from top to bottom, resulting in a redesigned workflow that lets users zoom into details within a project and zoom out to get an overview of their project portfolio with just two clicks. The result is happier users, fewer support tickets, and greater app use overall.
